如何安裝 boot-chart

Apr 24, 2016 | boot-chart | Hits
  1. 安裝 opne-jdk

  1. 確認安裝成功

  2. 下載並編譯 bootchart-lite

  3. 配置 linux 啟動參數
    init=/usr/sbin/bootchart-lite
    這樣 linux 啟動時就會將 bootchart-lite 啟動

  4. 使用 kill -10 [bootchart-lite ID] 結束 bootchart-lite

  5. 將 proc_diskstats.log, proc_ps.log, proc_stat.log 打包成 tgz 檔案

  6. 使用 java 生成 bootchart.png

    若顯示沒有找到 bootchart.png ，則需要手動建立一個

  7. 開啟 bootchart.png 即可顯示開機流程與效能分析
