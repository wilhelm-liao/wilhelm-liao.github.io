嵌入式系統如何設置 TimeZone

Sep 30, 2016 | Embedded Linux | Hits

  1. 在 etc 下建立 TZ 檔案，內容需填入時區，可參考 ubuntu 裡面 /usr/share/zoneinfo/ 的內容。
    國家: 時區
    Taipei: CST-8
    Hong_Kong: HKT-8
    Tokyo: JST-9

  2. hwclock & date 的說明
    以 UTC 12:00:00 & CST-8 說明

    • hwclock -u
      RTC 內容為 UTC 時間，會轉換成當地時間之後顯示，20:00:00

    • hwclock
      RTC 內容為當地時間，不需經過轉換就顯示，12:00:00

    • hwclock -u -w
      讀取 linux 的 UTC 時間，並寫入 RTC

    • hwclock -w
      讀取 linux 的 當地時間，並寫入 RTC

    • hwclock -u -s
      RTC 內容為 UTC 時間，轉換成當地時間之後寫入 linux

    • hwclock -s
      RTC 內容為當地時間，不需經過轉換就寫入 linux

    • date -u
      讀取 linux 的 UTC 時間，12:00:00

    • data
      讀取 linux 的本地時間，20:00:00

    • date -u -s 2016-01-01 12:00:00
      設定時間到 linux 的 UTC 時間，應用在 GPS 同步時間上

    • date -s 2016-01-01 12:00:00
      設定時間到 linux 的本地時間，應用在使用者直接設定當地時間
